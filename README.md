# cursorprueba

Proyecto clon de Notion con formulario de registro y autenticación Cloudflare Turnstile.

## Características

- Clon de la página principal de Notion
- Formulario de registro con validación dual (HTML + JavaScript)
- Integración con Cloudflare Turnstile para seguridad
- Diseño responsive y moderno

## Estructura del proyecto

```
mi-proyecto/
├── index.html
├── pages/
│   ├── registrarse.html
│   ├── logearse.html
│   └── acceso.html
├── css/
│   └── styles.css
├── js/
│   ├── main.js
│   ├── utils.js
│   └── api.js
├── assets/
│   ├── img/
│   ├── fonts/
│   └── icons/
└── vendor/
    └── normalize.css
```

## Tecnologías

- HTML5
- CSS3 (con variables CSS)
- JavaScript (ES6+)
- Cloudflare Turnstile
- Normalize.css
